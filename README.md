# Weather
![banner](https://cdn.modrinth.com/data/cached_images/057821c5d72f2209cf9db518d39737ad467e27a0.jpeg)
## INFO
Yun's Weather is a visual client-side mod for Minecraft:java, in which modifications to the original weather effects have been added to provide a refreshingly different look. It attempts to mimic real weather in a stylized, atmospheric, and aesthetic way, striking a balance between aesthetics and reasonableness.
Currently in open BETA, you may experience bugs, crashes and instability. I will implement more effects.

## FEATURES
* Particle system compatible with vs2,create
* Global Wind Impact Particle
* Rain on the wall generates splashes
* Dynamic biome color
* Dynamic biome sky color
* More to come!
## MODPACK

Weather can be used in whatever modpack you so desire, though please be aware of potential compatibility issues.

## Incompatible
* SImply Clouds
* Pretty Rain
